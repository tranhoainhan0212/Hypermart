Hypermart
   Hypermart là một nền tảng thương mại điện tử full-stack mà mình xây dựng từ đầu, bao gồm storefront động, xác thực đa kênh (JWT và Facebook OAuth), tích hợp thanh toán tự động qua VNPay/Momo, và một Admin Dashboard để quản lý tồn kho cũng như theo dõi đơn hàng theo thời gian thực.

Tech Stack
	Frontend: React, Tailwind CSS, Redux Toolkit, React Router
	Backend: Node.js, Express, MongoDB
	Thanh toán: VNPay, Momo API
	Xác thực: Facebook OAuth, JWT


Tính năng chính
	Đăng nhập bằng Facebook và xác thực JWT
	Giỏ hàng và luồng checkout đầy đủ
	Tích hợp cổng thanh toán VNPay và Momo
	Admin Dashboard quản lý sản phẩm và đơn hàng
	Giao diện responsive trên mọi thiết bị
